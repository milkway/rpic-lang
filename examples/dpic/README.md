# dpic example gallery

This directory contains examples from the **dpic** distribution, rendered to SVG
by **rpic** itself. They serve two purposes: a gallery of what rpic can draw, and
a parity check against the reference implementation of the *pic* language.

Many of the files under [`manual/`](manual) (the `man*.pic` set) are reproductions
of figures from the classic *pic* manuals — most originally by **Brian W.
Kernighan**, who created the *pic* picture-drawing language at Bell Labs. The
remaining files are diagnostic and demonstration diagrams from dpic.

Every `.pic` here is paired with the `.svg` that rpic produces from it.

## Rendering

```sh
# SVG (default), PNG, or PDF
rpic --svg examples/dpic/manual/man16.pic -o man16.svg
rpic --png examples/dpic/manual/man16.pic -o man16.png
rpic --pdf examples/dpic/manual/man16.pic -o man16.pdf
```

## Parity status

The current pass/fail matrix and known oracle quirks are tracked in
[`STATUS.md`](STATUS.md). In particular, this corpus preserves dpic source files
as oracle inputs, including SVG backend quirks such as `manual/man50` clipping
that are inherited from `dpic -v` rather than introduced by rpic.
